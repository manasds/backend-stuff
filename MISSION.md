# Mission: Build a Type-Safe Real-World API

## Why
Level up from basic CRUD to building a production-quality API with full type safety using Hono + Zod + OpenAPI + Drizzle ORM + PostgreSQL. Solidify Drizzle skills and learn API-first design with auto-generated docs.

## Success looks like
- A working API with multiple related entities, proper validation, and OpenAPI documentation
- Drizzle schema with relations, migrations, and efficient queries
- Type-safe request/response handling end-to-end
- Deployable with a real PostgreSQL database
- Ability to explain and reuse the patterns learned

## Constraints
- Use Hono + Zod + OpenAPI (scrict/* package)
- Use Drizzle ORM with PostgreSQL
- Want challenging but not overwhelming
- Have basic API experience
