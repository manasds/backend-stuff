# Recipe Manager API Resources

## Knowledge

- [Hono Documentation](https://hono.dev/)
  Official Hono docs. Use for: routing, middleware, context patterns.
- [@hono/zod-openapi Reference](https://github.com/honojs/middleware/tree/main/packages/zod-openapi)
  The `@hono/zod-openapi` package for generating OpenAPI specs from Zod schemas. Use for: route definitions, response schemas.
- [Drizzle ORM Docs](https://orm.drizzle.team/)
  Official Drizzle docs. Use for: schema definition, relations, queries, migrations.
- [Drizzle Kit](https://orm.drizzle.team/kit-docs/overview)
  Migration and push tooling for Drizzle. Use for: generating and running migrations.
- [Zod Documentation](https://zod.dev/)
  Schema validation. Use for: request validation, type inference.
- [OpenAPI 3.0 Specification](https://swagger.io/specification/)
  The spec behind OpenAPI. Use for: understanding what @hono/zod-openapi generates.
- [Article: "Building Type-Safe APIs with Hono and OpenAPI"](https://www.scaler.com/topics/hono-openapi/)
  Walkthrough of setting up Hono with OpenAPI. Use for: project structure patterns.

## Wisdom (Communities)

- [r/node](https://reddit.com/r/node)
  General Node.js community. Use for: debugging patterns, best practices.
- [Drizzle Discord](https://discord.gg/drizzle)
  Official Drizzle community. Use for: ORM-specific questions, migration issues.
